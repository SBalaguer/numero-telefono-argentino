# Validador de números de teléfono argentinos

## Ejemplos válidos
- +54 9 11 4639-1234
- 011 4639-1234
- 11 4639-1234
- 15 4639-1234
- 1550176006
- 4639.1234
- 4639-1234
- 911
- 0810 666 4444
- 000
- (54) 11 5789-1489
- (02966) 441200
